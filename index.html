التقدم في الدروس

الدرس 1: أول خطوة في Python

لغة Python سهلة القراءة وقوية جداً. أول أمر غالباً تتعلمه هو طباعة نص للمستخدم.

  • print('Hello World') يطبع النص.
  • input() يأخذ إدخال من المستخدم.
  • ابدأ بكتابة برامج صغيرة جداً حتى تثبت الفكرة.

    • الدرس 2: أنواع البيانات

    من أهم الأنواع: النصوص strings، الأعداد الصحيحة integers، الكسور floats، والقيم المنطقية booleans.

  • name = 'Ali'
  • age = 21
  • price = 19.5
  • is_active = True

    • الدرس 3: الدوال

    الدوال تساعدك تعيد استخدام الكود بدل تكراره.

  • def greet(name): return f'Hello {name}'
  • الدوال تستقبل parameters وترجع return.
  • كلما كبر مشروعك زادت أهمية تقسيمه إلى دوال.

    • الدرس 4: القوائم والقواميس

    القائمة تخزن عناصر مرتبة، والقاموس يخزن بيانات على شكل مفتاح وقيمة.

  • items = ['pen', 'book', 'phone']
  • user = {'name':'Ali', 'age':21}
  • تقدر تبحث وتضيف وتحذف وتعدل بسهولة.

    • الدرس 5: الجمل الشرطية

    الشرط يخلي البرنامج يتخذ قرار حسب المدخلات.

  • if age >= 18: السماح بالدخول
  • elif لعدة احتمالات
  • else للحالة الافتراضية

    • الدرس 6: التعامل مع الملفات

    التعامل مع الملفات ضروري لتخزين البيانات وقراءتها من نصوص خارجية.

  • with open('data.txt', 'r') as file:
  • content = file.read()
  • استخدم with حتى يغلق الملف تلقائياً.

    • الدرس 7: معالجة الأخطاء

    ليس كل شيء يعمل دائماً، لذلك Python توفر try / except للتعامل مع الأعطال.

  • try: جرب التنفيذ
  • except: التقط الخطأ
  • finally: أوامر نهائية

    • الدرس 8: OOP

    البرمجة الكائنية تنظم المشروع على شكل كائنات لها خصائص وسلوك.

  • class Car:
  • def __init__(self, brand):
  • أنشئ كائنات متعددة من نفس الفئة.

    • الدرس 9: الوحدات والمكتبات

    الموديولات تسمح لك بتقسيم البرامج، والمكتبات تضيف قدرات جاهزة.

  • import math
  • from datetime import datetime
  • استخدم المكتبات لتسريع التطوير.

    • الدرس 10: الأتمتة

    واحدة من أقوى استخدامات Python هي تنفيذ مهام متكررة بشكل تلقائي.

  • إعادة تسمية الملفات
  • تنظيم المجلدات
  • إنشاء تقارير تلقائية

    • الدرس 11: بناء المشاريع

    ابدأ دائماً بفكرة صغيرة، قسّمها إلى أجزاء، ثم ابْنِ واجهة منطقية نظيفة.

  • حدد الهدف
  • ارسم التدفق
  • ابنِ نسخة أولية MVP

    • الدرس 12: مدخل لتحليل البيانات

    هذا الباب يأخذك إلى مجالات أقوى مثل قراءة CSV وتحليل الجداول ورسم النتائج.

  • NumPy للحسابات
  • Pandas للجداول
  • Matplotlib للرسوم البيانية
